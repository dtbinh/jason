This Jason application exemplifies how to develop agent in a organised
society based on the Moise+ organisational model (http://moise.sf.net).

To run the application using JasonIDE, open the project 
    example/writePaper/writePaper.mas2j
and start it.

The chapter 4 of doc/tutorial.pdf explains in details this example.

Note: Jason 0.9.1 is required.
